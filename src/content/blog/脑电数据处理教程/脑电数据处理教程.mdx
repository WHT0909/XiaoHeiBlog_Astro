脑电图（Electroencephalography, EEG）通过放置在头皮表面的电极记录大脑神经元的突触后电位变化，是认知神经科学和临床诊断中广泛使用的非侵入式脑功能检测技术。由于头皮脑电信号强度微弱且由多种信号混合叠加而成，原始数据需要经过严格的预处理才能用于后续的统计分析。本文从脑电信号的生理基础出发，介绍脑电的预处理方法，并基于 EEGLAB（2026.0.0 版本）详细阐述数据导入、电极定位、滤波、降采样、分段与基线校正、插值坏导、重参考、独立成分分析（ICA）以及伪迹剔除等十个预处理关键步骤的具体操作。

## 1. 脑电信号的产生

大脑中的神经元通过动作电位和突触后电位进行信息传递，其电活动可通过不同尺度的记录技术进行测量：

- **膜片钳法**：在离体细胞或脑片上记录单个离子通道的电流活动，信号强度为毫伏（mV）级别，但无法在活体上进行。
- **局部场电位**：将微电极插入深部脑区，记录电极尖端附近神经元群体的突触后电位总和，信号强度为微伏（$\mu \rm{V}$）级别，属于侵入式记录。
- **头皮脑电**：在头皮表面放置电极，记录大脑皮层锥体神经元同步放电所产生的电场变化。由于信号须穿过颅骨、脑脊液和头皮等介质（即容积传导效应），其强度极为微弱，且由多种信号**混合叠加**而成。

本文所讨论的"脑电"指头皮脑电。

## 2. 脑电的特征与分析

脑电信号可从以下四个维度进行描述：

- **时间**：信号随时间的演变过程
- **幅度**：信号的电压强度（$\mu \rm{V}$）
- **频率**：单位时间内的振荡次数（Hz）
- **通道**：不同电极位置的信号空间分布

基于上述维度，脑电分析可分为以下几类：

- **时域分析**：时间 + 幅度，如事件相关电位的波幅和潜伏期测量
- **频域分析**：幅度 + 频率，通过傅里叶变换将时域信号转换至频域，计算各频段的功率谱密度
- **时频分析**：时间 + 幅度 + 频率，利用短时傅里叶变换（STFT）或小波变换考察信号的频谱随时间的动态变化
- **功能连接**：时间 + 幅度 + 频率 + 通道，分析不同脑区之间的相干性、相位同步等关系

根据实验范式的不同，脑电图可分为两类：

- **EEG（静息态）**：受试者在闭眼或睁眼静息状态下记录的自发脑电活动。主要用于频域分析、时域分析、功能连接和微状态分析，常作为被试间变量用于正常人与患者的对比研究。
- **ERP（任务态）**：事件相关电位，通过向受试者呈现特定刺激（视觉、听觉等）诱发的锁时脑电活动。主要用于时域分析、时频分析和溯源分析，常作为被试内变量用于不同实验条件的对比。

![EEG与ERP的分类](./imgs/Pasted_image_20260415161106.png)

脑电技术的核心优势在于其**高时间分辨率**（可达毫秒级），特别适合时频分析；但其空间分辨率较低，难以精确定位深部脑区的神经活动源。

### 3.1 导入数据

不同厂商的脑电采集设备输出的数据格式各不相同。常见的原始数据格式包括：

| 格式 | 说明 |
|------|------|
| `.eeg` + `.vmrk` + `.vhdr` | BrainVision 格式，三者配套，不可缺失 |
| `.edf` / `.bdf` | 欧洲标准格式，`.bdf` 精度更高 |
| `.cnt` | 老款 NeuroScan 设备导出格式 |
| `.mff` | EGI 设备导出格式 |

EEGLAB 处理的内部格式为 `.set` + `.fdt`。

<Aside>
**脑电数据格式**

BrainVision 格式（`.eeg` + `.vmrk` + `.vhdr`）中三个文件必须完整：
- `.vhdr`（头文件）：记录电极位置、采样率、通道数等元信息
- `.vmrk`（标记文件）：记录事件和刺激标记（如刺激出现的时间点）
- `.eeg`（数据文件）：存储真实的脑电波形数据

EEGLAB 格式（`.set` + `.fdt`）：
- `.set`：相当于说明书，记录通道信息、事件、采样率、电极位置和处理历史等信息
- `.fdt`：存储真实的脑电信号数据
</Aside>

1. 通过 `File → Load Existing Dataset` 加载已保存的 `.set` 文件：

### 3.4 滤波

脑电数据中混有来自环境和生理活动的各类噪声，包括 50 Hz 的市电（工频）干扰、高频肌电噪声和低频皮肤电漂移等。**滤波的目的并非直接在时域中将超过截止频率的信号"删除"，而是通过傅里叶变换将信号转换至频域，在频域中去除噪声对应的频率成分后再逆变换回时域**。

常用的四种滤波器：

- **高通滤波器**：保留高于截止频率的信号成分，抑制低频漂移
- **低通滤波器**：保留低于截止频率的信号成分，抑制高频噪声
- **带通滤波器**：仅保留特定频率范围内的信号
- **带阻滤波器**：抑制特定频率范围内的信号（如 50 Hz 工频干扰）

滤波参数的选取须结合脑电的自然节律：

![脑电节律分类表](./imgs/Pasted_image_20260415210953.png)

推荐参数范围：
- 低通滤波（上界）：30–100 Hz
- 高通滤波（下界）：0.1–1 Hz
- 带阻滤波（陷波）：49–51 Hz 或 48–52 Hz

### 3.5 降采样

采样率指单位时间内记录的采样点数量，单位为 Hz。其计算公式为：

$$\text{采样率} = \frac{\text{采样点个数}}{\text{采样时间}}$$

采样从时刻 0 开始：

![采样示意图](./imgs/Pasted_image_20260416164123.png)

根据 Nyquist 采样定理，采样率须大于感兴趣的信号最高频率的 2 倍，通常建议达到 3–4 倍。对于脑电信号而言，**250–1000 Hz 的采样率已足够**。过高的采样率会增大数据量，降低后续计算效率。降采样的意义在于**减少数据量、提高计算速度**，但此步骤并非必须。

**操作**：`Tools → Change sampling rate`

![降采样菜单](./imgs/Pasted_image_20260416165022.png)

将采样率设为 500 Hz：

![设置新采样率](./imgs/Pasted_image_20260416165056.png)

### 3.6 分段和基线校正

<Aside type="caution">
本步骤仅适用于 ERP（任务态）数据的处理流程。
</Aside>
静息态脑电存在基线漂移现象——脑电的直流电位会随时间缓慢漂移，导致不同试次的基线水平不一致。分段和基线校正的目的即是从连续数据中截取刺激相关的时间片段（Epoch），并以刺激前的一段基线期为参考进行校正。

![基线漂移示意图](./imgs/Pasted_image_20260416165818.png)

<br/>

![基线校正原理](./imgs/Pasted_image_20260416165933.png)

**分段操作**：`Tools → Extract epochs`

![Extract epochs菜单](./imgs/Pasted_image_20260416170046.png)

![分段参数设置](./imgs/Pasted_image_20260416170323.png)

<Aside>
**Time-locking event type(s) 参数说明**

此字段用于指定以何种事件编码为基准切分数据。若留空并按 OK，EEGLAB 将对数据中所有事件类型（`10`, `100`, `11`, `200`...）进行切分，生成混杂的试次；若输入特定事件编码（如 `10`），EEGLAB 仅在该事件出现的时间点前后进行切分。对于分条件的分析（如仅分析"正确反应"试次），正确选择事件类型至关重要。
</Aside>

**Out-of-bounds EEG limits if any [min max]** 是 EEGLAB 中处理事件靠近数据边界的安全限制参数，用来处理事件离数据边界太近，导致截取的试次超出原始数据范围的情况：
- 留空（默认）：若截取范围超出原始数据边界则直接报错，确保数据完整性
- 填写 `[min max]` 数值：允许生成不完整试次，超出部分用填充值替代，保证试次长度一致。比如填 `[-100 100]`，超出数据范围的部分会用 ±100 的数值填充（或 NaN），保证试次长度一致

点击`Time-locking event type(s)`右侧的`...`可查看数据中的全部事件类型：

![事件列表](./imgs/Pasted_image_20260416170352.png)

事件编码含义（本示例数据集）：
- `10`、`11`：分别为不同类型刺激的呈现编码
- `100`、`200`：分别为错误和正确反应的事件标记
- `boundary`：边界事件，通常标记数据分段处
- `empty`：空事件

<Aside>
`10`, `100`, `11`, `200`等是**数据中具体的事件编码**（这里是自己采的数据里设置的）。比如实验代码中，“10” 可能代表 “呈现红色刺激”，“200” 代表 “呈现绿色刺激”。
</Aside>

### 3.7 插值坏导和剔除坏段

<Aside type="caution">
**此步骤务必逐段手动检查，不可自动跳过。**
</Aside>
**坏导**指某一个电极通道的信号在整个记录过程中出现异常（如电极接触不良导致的持续噪声或信号丢失）。插值坏导使用 **Spherical 算法**，基于邻近正常电极的信号通过球面样条插值估计坏导通道的电位。

**容积传导效应**：大脑内部的电活动并非直接传导至头皮上的电极，而是经由脑脊液、颅骨和头皮等介质传播。正是由于这种信号在空间上的连续性，才能利用周围正常电极的数据对坏导通道进行数学插值。

**坏段**指部分时间段内所有通道信号均出现异常。与坏导不同，坏段直接删除即可——因为最终的 ERP 分析会对所有保留的试次进行叠加平均，个别坏段的删除不会影响结果的统计有效性。

![坏导坏段示意图](./imgs/Pasted_image_20260416201819.png)

**坏导插值操作**：

假设通道 O1 为坏导。通过 `Tools → Interpolate electrodes`，在 Channel 输入框中输入 O1，勾选 `on remove`，保存即可：

![选择O1通道](./imgs/Pasted_image_20260416202222.png)

![移除O1通道](./imgs/Pasted_image_20260416202338.png)

**去坏段操作**：

在脑电波形浏览界面，用鼠标左键选中坏段区域，点击下方的 `Reject` 按钮：

![手动选择坏段](./imgs/Pasted_image_20260416202758.png)

此时控制台输出 `2/100 trials rejected`，数据集的 epoch 数目由 100 减少为 98：

![去坏段后的数据](./imgs/Pasted_image_20260416202919.png)

### 3.8 重参考

脑电信号本质上是两个电极之间的电位差。在线记录时通常使用一个物理参考电极，而离线分析中常需要重新选择参考点（重参考），以满足不同分析目的。常见参考方案包括：

- **全脑平均参考**：将所有电极的平均电位作为参考，适用于高密度导联
- **双侧乳突参考**：以 TP9、TP10（双侧乳突电极）的平均电位为参考，常用于低密度导联
- **REST 参考**：基于溯源模型的零参考技术

**重参考必须置于插值步骤之后**，以确保参与平均参考计算的电极均为有效电极。

![重参考概念图](./imgs/Pasted_image_20260416203324.png)

**操作**：`Tools → Re-reference the data`

![重参考菜单](./imgs/Pasted_image_20260416203515.png)

![重参考设置](./imgs/Pasted_image_20260416203927.png)

<Aside>
**重参考方法**

- **Compute average reference**：全脑平均参考，将所有电极的均值作为新的参考零点
- **Huber average ref. with threshold**：基于 Huber 回归的进阶版平均参考，对异常电极具有更好的鲁棒性
- **Re-reference data to channels**：双侧乳突平均参考，选中的 TP9 TP10 是对应的乳突电极
- **Interpolate removed channel(s)**：插值移除的通道
</Aside>

重参考完成后，数据集的 `Reference` 字段显示为 `TP9 TP10`：

![重参考完成](./imgs/Pasted_image_20260416204524.png)

### 3.9 独立成分分析（ICA）

独立成分分析（Independent Component Analysis, ICA）是脑电预处理中的核心步骤，旨在从混合的头皮脑电信号中分离出统计上独立的源成分，从而识别并去除眼动、眨眼、肌电等伪迹。

ICA 的前提假设：
1. 头皮记录的脑电信号是多个源信号（脑活动源 + 伪迹源）的线性混合
2. 源信号之间在统计上相互独立
3. 源信号的数量不超过记录通道的数量

![ICA原理示意图](./imgs/Pasted_image_20260416204808.png)

![ICA分解示意图](./imgs/Pasted_image_20260416205207.png)

**操作**：`Tools → Decompose data by ICA`

![ICA菜单](./imgs/Pasted_image_20260416205329.png)

![ICA参数设置](./imgs/Pasted_image_20260416205540.png)

上图中 PCA 成分数设为 59，计算方法为：原始 64 通道 − 剔除 HEOG、VEOG（2 个）− 剔除双侧乳突参考（2 个）− 插值的坏导（1 个）= 59 个有效成分。ICA 在运行时将首先使用 PCA 将数据降至该维度。
